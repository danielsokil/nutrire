## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

Install gems with:

```bash
bundle install
```

Setup database with:

```bash
rails db:create
rails db:migrate
```

### Usage

Start server with:

```bash
rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```bash
rpsec --format documentation
```

### Lint

```bash
rails lint
```
